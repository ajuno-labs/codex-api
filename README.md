# Codex API

## Description

Laravel backend for Codex.

## Installation

1. Clone the repository
2. Run `composer install`
3. Run `php artisan key:generate`
4. Run `php artisan migrate`
5. Run `php artisan db:seed`
6. Run `php artisan serve`
